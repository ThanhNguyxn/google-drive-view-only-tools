## üöÄ Fastest Methods

### For Google Docs

**Option 1: One-Click Bookmarklet** (Recommended for repeated use)
1. Copy all content from `bookmarklet.js`
2. Create a new bookmark in your browser
3. Paste the code as the bookmark URL
4. Click the bookmark when viewing any Google Doc

**Option 2: Console Script** (One-time use)
1. Open the Google Doc
2. Press `F12` ‚Üí Click "Console" tab
3. Copy & paste content from `script.js`
4. Press Enter

### For Google Sheets

**Method:** Print to PDF or Copy Data
- Press `Ctrl+P` ‚Üí Select "Save as PDF"
- OR: Change URL from `/edit` to `/htmlview` ‚Üí Copy all data

### For Google Slides

**Method:** Print to PDF
- Press `Ctrl+P` ‚Üí Select "Save as PDF"
- Choose 1 slide per page for best quality
